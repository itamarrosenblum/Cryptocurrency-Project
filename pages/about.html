About

• Project

Cryptocurrency Data Feed.

• Decsription

Single pag application that provides information and real-time reports of cryptocurrencies.

• Environment

JavaScript, HTML, CSS.

• 3rd Party API Integration

Integrated 3rd party APIs for live pricing, coins information and images.

• User Story

  • Users can view in real-time prices of cryptocurrencies.
  • Users can save up to 5 cryptocurrencies to a report list.
  • Users can search for their favorite cryptocurrencies.
  • Users can manage (add/remove) their favorite cryptocurrencies (report list).
  • The website pages are created dynamically.
